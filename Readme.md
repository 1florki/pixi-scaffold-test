## Pixi scaffold

#### features:

- automatically scales your canvas
- toggle fullscreen
- gesture recognition: tap, doubletap, swipe, press, pan, rotate, pinch
- keydown and up event
- show/hide stats
- random seeded number generator
- easier way to draw/update shapes
- simple particle system
- vector class


#### future features:
